# Contributing

Thank you for your interest in improving Windows CleanSweep Toolkit.

## How to contribute

1. Fork the repository.
2. Create a new branch.
3. Test your changes on a non-production Windows machine.
4. Submit a pull request with a clear description.

## Contribution ideas

- Add more cleanup scan categories.
- Improve HTML reporting.
- Add support for scheduled cleanup reports.
- Add GUI wrapper.
- Add screenshots and demo GIFs.

## Safety rules

Please avoid destructive cleanup logic. Any automatic deletion must be conservative, documented and easy to review.
