# IT Support Cleanup Checklist

## Before cleanup

- Confirm the user's issue.
- Check free space on C: drive.
- Ask whether Downloads contains important files.
- Close browsers and Office apps.
- Run scan mode first.

## During cleanup

- Clean only temporary files first.
- Do not delete user documents.
- Do not delete Downloads automatically.
- Keep the generated report.

## After cleanup

- Restart the device if required.
- Confirm disk space improvement.
- Confirm the original issue is resolved.
- Document actions taken in the ticket.
